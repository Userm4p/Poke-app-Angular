Welcome to Poke App!

This is an application designed to explore and discover detailed information about Pokémon.

With Poke App you can:

  • 📚 Search Pokémon by name
  • 📊 View detailed stats for each Pokémon
  • ⚔️ Explore different types
  • 🌍 Learn about the Pokémon world regions
  • 💥 Explore moves and their effects

More features coming soon!
